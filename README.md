# SolarCalc LRV ☀️
**Projeto de Extensão - Seminário Integrador IV (BCT/UFMT - 2026)**

Este é um Web App desenvolvido em Python com Streamlit para calcular a viabilidade técnica e financeira de sistemas de energia solar fotovoltaica para a comunidade de **Lucas do Rio Verde/MT**.

## 🎯 Objetivo
Auxiliar moradores e pequenos comerciantes locais na transição energética, fornecendo um diagnóstico preciso baseado em dados reais de irradiância da região.

## 📚 Articulação Acadêmica
O projeto integra os conhecimentos das seguintes disciplinas:
* **Física III:** Modelagem do efeito fotovoltaico e perdas energéticas no sistema.
* **Cálculo III:** Otimização de área e fluxo de geração.
* **Probabilidade e Estatística:** Análise de dados climáticos e índices de radiação solar (HSP).
* **Matemática Financeira:** Cálculos de Payback, VPL e retorno sobre investimento.

## 🚀 Tecnologias Utilizadas
* Python 3.x
* Streamlit (Interface Web)
* Pandas & NumPy (Processamento de dados)
* Matplotlib/Plotly (Visualização de dados)
